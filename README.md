# SleepLog Store Screenshots

Auto-generated iPhone 17 Pro screenshots for the SleepLog App Store listing.

**Source**: private repo `Forte-design/app-sleep-log` — `scripts/capture-screenshots.sh` + `-ScreenshotScreen` launcher.
**Purpose**: public mirror so Figma `createImageAsync` can fetch via raw URLs when composing store mockups.

## Layout

```
{asc-locale}/
├── 01_recording.png  (SleepView - recording state)
├── 02_detail.png     (SessionDetailView - hero session)
└── 03_list.png       (ReportView - 1W period)
```

34 ASC locales × 3 screens = 102 PNGs. Device: iPhone 17 Pro (1206 × 2622). Status bar overridden to 9:41 / full wifi / charged battery.
